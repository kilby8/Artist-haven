Update Guide for BlueStacks

💡 Tip: BlueStacks on Windows can access your local network, so you can download directly from the server!

📱 Method 1: Browser Download (RECOMMENDED)

Most convenient! Download directly in BlueStacks from your Windows computer.

  1. Start the server: python3 apk_server.py
  2. Open BlueStacks and launch its browser
  3. Visit: http://localhost:8000/DOWNLOAD_APK.html
    (or use your computer IP if on different network segment)
  4. Click the download button - APK downloads to BlueStacks downloads
  5. Open your file manager in BlueStacks
  6. Navigate to Downloads folder
  7. Tap the APK file to install
  8. Grant permissions and confirm installation
✓ App installs automatically, ready to use!

⚙️ Method 2: ADB Command (FASTEST UPDATE)

For rapid iterations! One command uninstalls old version and installs new one.

Setup (First Time Only):

  1. Enable ADB on BlueStacks:
    BlueStacks Settings → Advanced → Enable Android Debug Bridge
  2. Connect ADB:
    adb connect 127.0.0.1:5555
  3. Verify connection:
    adb devices (should show your BlueStacks device)

Update Command:

Run this each time you build a new APK:

adb uninstall com.artisthaven.app && adb install app/build/outputs/apk/release/app-release.apk
✓ Old version removed, new version installed - done in seconds!

🎯 Method 3: Drag & Drop (EASIEST)

Simple file transfer - Just drag the APK into BlueStacks.

  1. Open the APK location:
    app\build\outputs\apk\release\app-release.apk
  2. Have BlueStacks window open
  3. Drag the APK file into BlueStacks window
  4. BlueStacks will prompt to install - tap Install
  5. Wait for installation to complete
⚠️ Note: This may not uninstall the old version first. You might need to manually uninstall the old app before installing a new one if there are conflicts.

📦 Method 4: BlueStacks App Manager

Using BlueStacks UI - Install from local files.

  1. In BlueStacks: Click Menu (top right) → Install APK
  2. Browse to: app\build\outputs\apk\release\app-release.apk
  3. Click Open - BlueStacks installs automatically
  4. App appears in your apps - tap to launch
💡 Pro Tip: You can create a shortcut on your desktop to this APK file for quick re-installation!

🚀 Recommended Development Workflow

For rapid testing during development:

  1. Build APK in Android Studio
  2. Run one-line ADB command:
    adb uninstall com.artisthaven.app && adb install app/build/outputs/apk/release/app-release.apk
  3. Test immediately in BlueStacks
  4. Make code changes and repeat

⏱️ Total time per update: ~10-15 seconds

🔄 Bonus: Automated Update Script

Windows batch file - Build and install in one click!

Create a file called update-bluestacks.bat in your project root:

@echo off
echo Building APK...
call gradlew.bat assembleRelease

if errorlevel 1 (
  echo Build failed!
  exit /b 1
)

echo.
echo Uninstalling old version...
adb uninstall com.artisthaven.app

echo.
echo Installing new version...
adb install app\build\outputs\apk\release\app-release.apk

if errorlevel 1 (
  echo Installation failed!
  exit /b 1
)

echo.
echo ✓ Update complete!
pause

Usage: Just double-click update-bluestacks.bat and wait for the prompt!

🔄 Complete Reinstall (If Something Goes Wrong)

  1. In BlueStacks: Long-press on Artist Haven app
  2. Select Uninstall and confirm
  3. Or use ADB:
    adb uninstall com.artisthaven.app
  4. Then install fresh:
    adb install app/build/outputs/apk/release/app-release.apk

⚠️ Troubleshooting

ADB connection fails

  • Check BlueStacks ADB is enabled (Settings → Advanced)
  • Try: adb connect 127.0.0.1:5555
  • Run: adb devices to verify

Installation fails with "App not installed"

  • Uninstall old version first: adb uninstall com.artisthaven.app
  • Make sure APK file exists at the path
  • Try installing again

App crashes on launch

  • Clear app data: adb shell pm clear com.artisthaven.app
  • Or just uninstall and reinstall fresh
