📋 Installation Instructions

  1. Enable Unknown Sources: Go to Settings → Security → Unknown Sources (or Developer Options) and toggle ON
  2. Download APK: Click the download button above or scan the QR code
  3. Open Downloads: Find the APK file in your Downloads folder
  4. Install: Tap the APK file and confirm installation
  5. Grant Permissions: Allow necessary permissions when prompted
  6. Launch: Open Artist Haven from your apps drawer

🔨 Build Locally (Alternative)

  1. ./gradlew assembleRelease - Build release APK
  2. APK located at: app/build/outputs/apk/release/app-release.apk
  3. Transfer to device via USB or ADB
  4. Or: ./gradlew installRelease with device connected
